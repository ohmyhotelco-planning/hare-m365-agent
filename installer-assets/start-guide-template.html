이 화면에서 도메인 허용, 로그인, Cowork 프롬프트까지 순서대로 진행하면 됩니다.

1Claude/Cowork 도메인 허용

Claude/Cowork가 Microsoft Graph에 접속할 수 있도록 설정에서 도메인 2개를 허용합니다.

  1. Claude 앱 왼쪽 아래 계정/조직명(예: CTO Office · Max)을 클릭하고 설정을 엽니다.
  2. 왼쪽 설정 메뉴에서 기능을 클릭합니다.
  3. 오른쪽에서 네트워크 접근 섹션을 찾습니다.
  4. 도메인 허용 목록 아래 추가 허용 도메인 입력칸에 아래 도메인을 하나씩 입력하고 추가를 누릅니다.
graph.microsoft.com
login.microsoftonline.com
두 도메인이 목록에 보이면 완료입니다. 이 설정이 없으면 Cowork에서 network_error, proxy, 403 오류가 날 수 있습니다.

2최초 1회 로그인

가장 쉬운 실행 방법: 키보드의 윈도우키를 누르고 아래 경로를 그대로 붙여넣은 뒤 Enter를 누릅니다.

__ESCAPED_LOGIN_CMD__

또는 설치 폴더에서 로그인 시작.cmd 파일을 직접 실행해도 됩니다.

로그인 창에서 해야 할 일

  1. 검은 명령 창이 열리고 Microsoft 로그인 페이지가 자동으로 열립니다.
  2. 검은 명령 창에 device code가 표시됩니다. 아래 예시처럼 문장 끝에 보이는 짧은 코드입니다.
  3. 브라우저의 Microsoft device login 화면에 그 코드를 직접 입력합니다.
  4. 회사 Microsoft 계정으로 로그인합니다.
  5. 로그인이 끝나면 검은 창으로 돌아와 loggedIn: true가 보이는지 확인합니다.
To sign in, use a web browser to open the page https://login.microsoft.com/device
and enter the code 화면에 표시된 코드 to authenticate.
device code는 일회성 로그인 코드입니다. Claude/Cowork/채팅에 붙여넣지 말고, 사용자가 브라우저에 직접 입력하세요.

브라우저가 자동으로 열리지 않으면 https://microsoft.com/devicelogin을 직접 열고 검은 창의 코드를 입력합니다.
