2최초 1회 로그인

Finder에서 이 폴더를 열고 아래 파일을 더블클릭합니다.

__ESCAPED_LOGIN_CMD__

파일 이름은 로그인 시작.command입니다.

더블클릭으로 열리지 않으면 파일을 Control-클릭 또는 오른쪽 클릭한 뒤 열기를 선택하세요. 확인 창이 뜨면 다시 열기를 누릅니다. 회사 보안 정책상 계속 차단되면 IT 배포 담당자에게 Jamf/Intune 배포본을 요청하세요.
  1. 터미널 창이 열리고 Microsoft 로그인 페이지가 자동으로 열립니다.
  2. 터미널 창에 device code가 표시됩니다.
  3. 브라우저의 Microsoft device login 화면에 그 코드를 직접 입력합니다.
  4. 회사 Microsoft 계정으로 로그인합니다.
  5. 로그인이 끝나면 터미널에서 loggedIn: true가 보이는지 확인합니다.
To sign in, use a web browser to open the page https://login.microsoft.com/device
and enter the code 화면에 표시된 코드 to authenticate.
device code는 Claude/Cowork/채팅에 붙여넣지 말고, 사용자가 브라우저에 직접 입력하세요.
