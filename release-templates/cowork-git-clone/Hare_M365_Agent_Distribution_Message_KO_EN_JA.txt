3. Follow the guide to select a project folder, allow the required domains, complete the initial connection, and sign in to Microsoft.
4. After the connection is ready, ask for Outlook, Teams, or SharePoint/OneDrive information in natural language within the same Cowork chat.

When signing in to Microsoft, use your own company Microsoft account that you will use with Hare, not an account shown in an example screenshot.

[日本語]

こんにちは。

Claude/CoworkからMicrosoft 365の業務データを活用できるHare M365 Agentとスタートガイドを配布します。

Hare M365 Agentを使用すると、ご自身の会社MicrosoftアカウントでClaude/CoworkからOutlook、Teams、SharePoint/OneDriveの情報を参照し、必要な内容を整理できます。ユーザーの承認後にOutlookメールの下書きを作成することもできますが、送信はOutlookからご自身で行ってください。

Session 4のMicrosoft 365 MCP演習を行う場合は、MCPの代わりにHare M365 Agentを使用してください。

初めて使用する場合は、以下の手順で進めてください。

1. 添付された日本語版スタートガイドのHTMLファイルをPCにダウンロードします。
2. ダウンロードしたファイルをEdgeまたはChromeで開きます。TeamsまたはOneDriveのプレビューではコピーボタンが動作しません。
3. ガイドに従って、プロジェクトフォルダーの選択、必要なドメインの許可、初回接続、Microsoftへのサインインを完了します。
4. 接続完了後、同じCoworkチャットでOutlook、Teams、SharePoint/OneDriveの情報を自然な言葉で依頼します。

Microsoftへのサインインには、サンプル画面に表示されているアカウントではなく、Hareで使用するご自身の会社Microsoftアカウントを使用してください。
